Componentes


Os componentes permitem que você crie rapidamente uma interface para seu aplicativo. O Front-MP vem com vários componentes, incluindo modais, pop-ups e cartões. Confira os exemplos abaixo para ver como cada componente se parece e aprender como usar cada um deles. Quando estiver familiarizado com os princípios básicos, vá até os exemplos para ver como seu projeto pode ficar.
